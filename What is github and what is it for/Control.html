O que é Git? um Sistema de Controle de Versão:

Apenas poder acessar o código dos outros colaboradores não é suficiente.

Precisamos manter o histórico dos nossos arquivos.
Para isso, usamos o Git.
Git é um sistema de controle de versão, que permite aos desenvolvedores manter um histórico de alterações em seus arquivos.
Mais: das nossas modificações, pois muitas vezes mudamos arquivos em grupo, num movimento único (um commit).
Dessa forma, podemos voltar atrás e recuperar o estado do sistema como ele era ontem, ou no ano passado, e comparar as mudanças, para encontrar bug, para estudar otimizações.

Todos nossos arquivos, assim como seus históricos, ficam em um repositório e existiam vários sistemas que gerenciavam repositórios assim, como CVS e SVN.>O Git é uma alternativa que tem um funcionamento mais interessante ainda: ele é distribuído e todo mundo tem uma cópia inteira do repositório, não apenas o "servidor principal".
Isso trás grandes vantagens que você não precisa aprender nesse exato momento.

O GitHub é um sistema de gestão de repositórios, que permite aos desenvolvedores criar repositórios e colaborar com eles.
    1. Para usar o Git, você precisa instalar o Git no seu computador.
    2. Para isso, você precisa ter um GitHub.
    3. Para usar o GitHub, você precisa ter um conta no GitHub.
    4. Para isso, você precisa ter um nome de usuário e senha.
    5. Para criar uma conta, basta acessar o GitHub e clicar em Sign up.
    6. Para entrar na sua conta, basta acessar o GitHub e clicar em Sign in.
    7. Para criar um repositório, basta acessar o GitHub e clicar em New repository.
    8. Para acessar o repositório, basta acessar o GitHub e clicar em Repositories.
